## SQL 코드카타

#### 1. 이름이 있는 동물의 아이디
```sql
SELECT ANIMAL_ID
FROM ANIMAL_INS
WHERE NAME IS NOT NULL
ORDER BY ANIMAL_ID
;
```

#### 2. 역순 정렬하기
```sql
SELECT
    NAME
    , `DATETIME`
FROM ANIMAL_INS
ORDER BY ANIMAL_ID DESC
;
```

#### 3. 중복 제거하기
```sql
SELECT COUNT(DISTINCT NAME) AS count
FROM ANIMAL_INS
WHERE NAME IS NOT NULL
;
```

#### 4. 동물의 아이디와 이름
```sql
SELECT
    ANIMAL_ID
    , NAME
FROM ANIMAL_INS
ORDER BY ANIMAL_ID
;
```

#### 5. 동물 수 구하기
```sql
SELECT COUNT(*) AS count
FROM ANIMAL_INS
;
```
